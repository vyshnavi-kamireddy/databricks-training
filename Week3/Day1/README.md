# 📊 Week 3 • Day 1 | SQL NULL Functions – Basics

> *Understanding how SQL handles missing values using NULL-related functions to write cleaner and more reliable queries.*

---

# 🎯 Objective

Learn how to identify and handle missing data in SQL using `IS NULL`, `IS NOT NULL`, and `ISNULL()`. These concepts are essential for writing robust queries that can deal with incomplete or unknown values in real-world databases.

---

# 🚀 What I Implemented

### 🔹 NULL Checking
- Retrieved employees with missing salaries using `IS NULL`
- Identified orders with available discounts using `IS NOT NULL`
- Retrieved products with missing categories
- Counted employees without assigned managers

### 🔹 Handling NULL Values
- Replaced missing salaries using `ISNULL()`
- Assigned default bonus values where bonuses were missing
- Displayed default order amounts for NULL values
- Replaced NULL product stock values with zero
