## 🚀 What I Implemented

### 🔹 Column Creation & Conditional Logic
- Added constant values using `lit()`
- Created conditional columns using `when()` and `otherwise()`
- Categorized employees based on salary, age, experience, and department

### 🔹 String Transformations
- Extracted values using `substring()`
- Replaced text using `regexp_replace()`
- Filtered records using `LIKE`
- Applied membership filtering with `isin()`

### 🔹 Range-Based Filtering
- Retrieved records using `between()`
- Filtered employees by salary, age, joining year, and employee ID ranges

### 🔹 Data Reshaping
- Created summary tables using `pivot()`
- Converted pivoted data back into rows using `unpivot()`
- Normalized datasets using `stack()`

### 🔹 Window Functions
- Ranked employees using `rank()`
- Assigned dense rankings with `dense_rank()`
- Generated sequential numbering using `row_number()`
- Retrieved previous and next values using `lag()` and `lead()`
