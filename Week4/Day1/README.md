# 🚀 What I Implemented

### 🔹 DataFrame Creation
- Created an Employee DataFrame using PySpark
- Defined schema and loaded sample employee records
- Explored the dataset using DataFrame operations

### 🔹 Data Selection
- Selected required columns using `select()`
- Retrieved employee information based on business requirements
- Displayed specific attributes for analysis

### 🔹 Data Filtering
- Applied `filter()` and `where()` transformations
- Filtered records using salary, department, city, age, and gender
- Combined multiple filtering conditions

### 🔹 Column Transformations
- Added new calculated columns using `withColumn()`
- Renamed columns with `withColumnRenamed()`
- Created conditional columns using `when()`

### 🔹 Data Cleaning
- Removed unnecessary columns using `drop()`
- Retrieved unique values using `distinct()`
- Eliminated duplicate records using `dropDuplicates()`

### 🔹 Sorting & Ordering
- Sorted records in ascending and descending order
- Performed multi-column sorting
- Ordered employee data for reporting

### 🔹 Aggregations
- Grouped data using `groupBy()`
- Calculated total, average, minimum, and maximum values
- Generated summary reports using aggregate functions
