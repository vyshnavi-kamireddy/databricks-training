## 🚀 What I Implemented

### 🔹 SQL Joins Assignment
- Retrieved student-course enrollment details using joins
- Identified students without course enrollments
- Found courses without assigned instructors
- Retrieved departments and instructors with missing relationships
- Generated reports combining students, courses, and instructors
- Implemented `LEFT JOIN`, `RIGHT JOIN`, `FULL OUTER JOIN`, and `CROSS JOIN`

### 🔹 Window Functions
- Ranked employees using `ROW_NUMBER()`, `RANK()`, and `DENSE_RANK()`
- Retrieved Top-N employees based on salary
- Applied `PARTITION BY` for department-wise ranking
- Calculated running totals and cumulative sales
- Used `LAG()` and `LEAD()` to compare previous and next order values

### 🔹 Analytical SQL
- Department-wise salary analysis
- Employee ranking and performance reporting
- Time-based sales analysis
- Customer order comparisons using window functions
